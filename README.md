# EOS (Experimental Operating System)

EOS is an educational operating system for x86 platform

Current progress:
----------------

- GDT and IDT setup
- Text terminal (80x25)
- PS/2 keyboard support
- Timer support
- RAM detection and Physical Memory Manager
- Very simple kernel command shell (KSH)
